# Media Log - Project Instructions

## 作業ルール

- 変更を加えたら必ずコミットしてプッシュする
- コミットメッセージは変更内容を簡潔に記述する

## プロジェクト構成

- `docs/` - GitHub Pages用のWebアプリ
  - `app.js` - メインのJavaScriptコード
  - `index.html` - HTMLとCSS
- `schedule.json` - データファイル（GitHub経由で読み書き）

## メディアタイプ

movie, anime, drama, tv, comedy, game, book, manga, radio, streaming, youtube
