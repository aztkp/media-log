# my-radiko

radiko用の個人ツール群

## 機能

### Chrome拡張 (`extension/`)

radikoのタイムフリー・ライブ再生を強化する拡張機能

- **スキップボタン**: -30s / -10s / +10s / +30s
- **シークバー**: フル幅スライダー、分数メモリ付き
- **倍速再生**: 0.5x / 1x / 1.5x / 2x
- **曲情報表示**: 現在再生中の曲名・アーティスト・アートワーク
- **曲リスト**: 番組内の全曲を表示、クリックでジャンプ
- **聴取ログ**: メモ付きで保存、Markdownエクスポート

#### インストール

1. `chrome://extensions` を開く
2. 「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」
4. `extension` フォルダを選択

#### キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl+.` (Mac: `Ctrl+.`) | 10秒進む |
| `Ctrl+,` (Mac: `Ctrl+,`) | 10秒戻る |

### 聴取ログ (`logs/`)

聴いたラジオの記録をカレンダー形式で管理

- 日付別のMarkdownファイル
- 番組名、曲リスト、メモを記録
